## 框架选用分析

1. vue(50kb)

  运行时代码过大、vue2和vue3运行时冲突

2. react(128kb)

  运行时代码过大，导入不同版本的react也可能会冲突

3. lit(15kb) `谷歌` `start: 17.2k`

  需要学习成本

4. preact(11kb) `start: 35.8k`

  preact，运行时小，学习成本低，与vue3/vue2/react不冲突

## 应用场景

> preact开发组件，在vue2/vue3项目中共同使用
